Installation¶

Scopes can be acqusired using either ONIX or Miniscope DAQ hardware, each of which have their own Bonsai library.

  1. First you will need to install Bonsai

  2. Open Bonsai. When you are greeted by the start menu, click “Manage Packages.”

  3. Set the package source to “Bonsai Community”.

  4. If you want to use ONIX hardware, to acquire from your scope, search for “ONIX” and install.

    Tip

    It is recommended to also install Bonsai.ONIX.Design because it provides a substantial set of data visualation and graphical hardare configuration elements.

  5. If you want to use the Miniscope DAQ Box to acqusire form your acope, search for Search for “Miniscope” and install.
