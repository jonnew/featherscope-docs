Software Guide¶

Featherscopes use Bonsai for acquisition. There are two libraries that can be used depending on what hardware is used to acquire data from your scope.

  1. Bonsai.ONIX - Bonsai library for the ONIX acqusition system.

  2. Bonsai.Miniscope - Bonsai library for the Miniscope DAQ.

Using this Documentation¶

This documentation contains functional workflows and operators. You can download workflows by clicking the icon. Alternatively, you can copy and paste them into the Bonsai Workflow Editor by clicking the icon and using Ctrl+v or equivalent in the editor:
